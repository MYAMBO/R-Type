## Quick Start for Contributors

### Prerequisites
*   **OS**: Linux (Main target), Windows supported.
*   **Compiler**: GCC 11+ or Clang 14+ (C++20 required).
*   **Build System**: CMake 3.20+.
*   **Dependencies**: SFML 3.0+.

### Build Instructions
```bash
git clone https://github.com/MYAMBO/R-Type.git
cd R-Type
mkdir build && cd build
cmake .. && make
```

## Contribution Guidelines

- Branches: Use dev for integration. Feature branches must be named feat/feature-name.
- Commits: We use Conventional Commits (e.g : feat(Dir): feature implemented)
